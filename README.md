# George-Stationery
A custom stationery website built with HTML/CSS, hosted on Netlify and synchronized with Github for continous deployment
## 🚀 Live Demo
View the live site here! https://papastationery.netlify.app/

## ✨ Features
* Responsive layout for all devices
* Clean and modern UI for stationery products
* Fast loading times via Netlify

## 🛠️ Tech Stack
* **HTML5** & **CSS3**
* **Netlify** for deployment
